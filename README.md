## Development
Want to contribute? Great! Make me a pull request.

### Run local
```sh
$ git clone https://github.com/Nicolas-alt/Twitter-clone && cd Twitter-clone
$ npm i
$ yarn start 
```
It's works on 
```
127.0.0.1:3000
```

#### Project dependencies 
- react-router-dom
- react-helmet
- node-sass
